# Project Foundation

## Overview

The **Project Foundation** documents establish the purpose, direction, and boundaries of Smart NUB Campus.

These documents answer the fundamental questions:

- What are we building?
- Why are we building it?
- What problems does it solve?
- What are the project's goals?
- What is included and excluded?
- Who are the stakeholders?

This section should be read before moving to the Requirements or System Design documentation.

---

# Documents

| Document              | Purpose                                                                        |
| --------------------- | ------------------------------------------------------------------------------ |
| Project_Overview.md   | Introduces Smart NUB Campus and its purpose.                                   |
| Problem_Statement.md  | Defines the academic problems the platform aims to solve.                      |
| Vision_and_Mission.md | Describes the long-term direction and guiding principles of the project.       |
| Objectives.md         | Lists the primary and secondary objectives of the platform.                    |
| Scope.md              | Defines the boundaries of the first version of the project.                    |
| Stakeholders.md       | Identifies the individuals and groups involved in or affected by the platform. |

---

# Reading Order

For the best understanding of the project, read the documents in the following order:

1. Project Overview

3. Vision and Mission
4. Objectives
5. Scope
