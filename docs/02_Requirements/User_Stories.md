# User Stories

## Overview

This document captures the needs of Smart NUB Campus (SNC) users from their perspective.

Each user story follows the standard format:

> **As a _[role]_, I want _[goal]_ so that _[benefit]_.**

These stories help translate user needs into system requirements and guide future implementation.

---

# Guest

### Registration

- As a guest, I want to create an account so that I can join the Smart NUB Campus community.
- As a guest, I want to submit my student information for verification so that my identity can be confirmed.
- As a guest, I want to log into my account after registration so that I can check my verification status.

---

# Student

## Identity

- As a student, I want to complete my profile so that other students can know my academic background and interests.
- As a student, I want to update my profile information so that it remains accurate.

---

## Learning

- As a student, I want to upload academic resources so that I can help other students.
- As a student, I want to browse academic resources so that I can learn from previous materials.
- As a student, I want to search for notes, assignments, and previous questions so that I can quickly find relevant study materials.
- As a student, I want to bookmark useful resources so that I can access them later.
- As a student, I want to ask the AI assistant academic questions so that I can better understand difficult concepts.
- As a student, I want the AI assistant to summarize study materials so that I can revise more efficiently.
- As a student, I want the AI assistant to help me prepare study plans so that I can manage my learning effectively.

---

## Community

- As a student, I want to ask academic questions so that other students can help me.
- As a student, I want to answer questions so that I can share my knowledge.
- As a student, I want to participate in department discussions so that I can stay informed.
- As a student, I want to comment on discussions so that I can contribute to conversations.

---

## Network

- As a student, I want to search for students by skills and interests so that I can find suitable collaborators.
- As a student, I want to discover students from different departments so that I can expand my academic network.
- As a student, I want to send connection requests so that I can build professional relationships.
- As a student, I want to include a short note with my connection request so that the recipient understands why I want to connect.
- As a student, I want to accept or decline connection requests so that I can manage my network.

---

## Collaboration

- As a student, I want to create project or research teams so that I can collaborate with others.
- As a student, I want to join existing teams so that I can participate in academic projects.
- As a student, I want to recruit teammates with specific skills so that my team has the expertise it needs.

---

## Communication

- As a student, I want to privately message my accepted connections so that I can communicate without using external social media.
- As a student, I want to receive notifications so that I know when someone interacts with me or my content.

---

## Reputation

- As a student, I want to earn reputation points for positive contributions so that my efforts are recognized.
- As a student, I want to earn badges so that my achievements are visible to the community.

---

# Administrator

## Verification

- As an administrator, I want to review verification requests so that only genuine NUB students gain access.
- As an administrator, I want to approve or reject verification requests so that platform integrity is maintained.
- As an administrator, I want to provide a rejection reason so that students understand how to correct their submission.

---

## Moderation

- As an administrator, I want to review reported content so that inappropriate material can be removed.
- As an administrator, I want to suspend user accounts when necessary so that the community remains safe.
- As an administrator, I want to restore suspended accounts after review so that users can regain access when appropriate.

---

## Platform Management

- As an administrator, I want to publish announcements so that important information reaches all students.
- As an administrator, I want to view platform statistics so that I can monitor platform growth and activity.

---

# Story Prioritization

## Must Have

- Registration and verification
- Student profiles
- Resource sharing
- AI academic assistant
- Student networking
- Team finder
- Direct messaging
- Discussions
- Q&A
- Administrator verification

---

## Should Have

- Notifications
- Reputation system
- Badges
- Resource bookmarks

---

## Could Have

- AI-assisted verification
- OCR-based student ID extraction
- Advanced recommendation system
- Faculty accounts
- Alumni accounts
