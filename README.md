# Smart NUB Campus Documentation

> Official documentation for Smart NUB Campus

---

## About

Smart NUB Campus is an AI-powered academic collaboration network exclusively for Northern University Bangladesh students.

This repository contains all project documentation, including requirements, system design, architecture, diagrams, UI/UX planning, and development guidelines.

---

## Repository Structure

- `docs/`: Contains project documentation files.
- `01_Project_Foundation`: Contains foundational information about the project.
- `02_Requirements`: Contains project requirements and specifications.
- `03_System_Design`: Contains the system design and architecture of the project.
- `04_Diagrams`: Contains diagrams related to the project.
- `05_UI_UX`: Contains UI/UX planning and design files.
- `06_Development`: Contains development guidelines and code-related documentation.
